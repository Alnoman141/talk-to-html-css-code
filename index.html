To test this app you must use Google Chrome.
        

        Commands available:
          - HTML open [tag]
          - HTML content [sentence]
          - HTML close
          - CSS open [tag]
          - CSS style [property-name] is [value] (i.e CSS style color is green)
          - CSS close
        

        For example, to make this page red, say:
          - CSS open body
          - CSS style background color is red
          - CSS close
